    Discrete-Time Linear Systems #

    This module formalizes discrete-time linear systems and proves asymptotic stability results.

    Main Definitions #

    • DiscreteLinearSystemState: Structure representing a discrete-time linear state-space system
    • system_evolution: Computes the state trajectory of the system
    • state_system_equation: Defines the state evolution equation x(k+1) = Ax(k) + Bu(k)

    Main Theorems #

    • asymptotic_stability_discrete: If the spectral radius of the system matrix A is less than one and the input is zero, then the state converges to zero asymptotically.

              theorem asymptotic_stability_discrete {σ : Type u_1} {ι : Type u_2} [NormedAddCommGroup σ] [NormedSpace ℂ σ] [NormedAddCommGroup ι] [NormedSpace ℂ ι] [CompleteSpace σ] (sys : DiscreteLinearSystemState σ ι) (h_init : sys.x 0 = sys.x₀) (h_state : state_system_equation sys) (h_zero_input : sys.u = zero_input) (h_spectral : spectral_radius_less_than_one sys.a) :
              Filter.Tendsto (fun (k : ℕ) => ‖sys.x k‖) Filter.atTop (nhds 0)

              Asymptotic Stability Theorem for Discrete-Time Linear Systems

              If the spectral radius of the state transition matrix A is strictly less than one, and the input is identically zero, then the state converges to zero as time goes to infinity.

              Mathematically: If ρ(A) < 1 and u(k) = 0 for all k, then ‖x(k)‖ → 0 as k → ∞.

              Proof Strategy #

              The proof combines several key results:

              1. Gelfand's Formula: The spectral radius equals lim sup ‖A^n‖^(1/n)
              2. Zero-input evolution: When u = 0, we have x(k) = A^k · x₀
              3. Exponential decay: ρ(A) < 1 implies ‖A^k‖ decays exponentially
              4. Operator norm bound: ‖x(k)‖ = ‖A^k · x₀‖ ≤ ‖A^k‖ · ‖x₀‖
              5. Squeeze theorem: Since ‖A^k‖ → 0, we get ‖x(k)‖ → 0

              This is the fundamental stability result for autonomous discrete-time linear systems.
